Installation
============

From PyPI (recommended):

.. code-block::

   pip install mckit-nuclides

With package manager (as dependency):

.. code-block::

   poetry add mckit-nuclides

From source:

.. code-block::

   pip install https://github.com/MC-kit/mckit-nuclides.git
